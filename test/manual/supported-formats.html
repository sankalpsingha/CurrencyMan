Number Formats

The extension supports various number formats:

  • Integers: $100, €50
  • Decimals: $10.99, €15.50
  • Thousands separators (commas): $1,000, €1,500.75
  • Large numbers: $1,000,000, €1,000,000.00

Note: The extension currently supports comma (,) as the thousands separator and period (.) as the decimal separator. Other formats like space as thousands separator or comma as decimal separator are not currently supported.

Spacing Variations

The extension handles various spacing between currency and amount:

  • No space: $100, 100€
  • Single space: $ 100, 100 €
  • Code with space: USD 100, 100 EUR

Limitations

The extension has some limitations to be aware of:

  • Currency symbols that are also used in other contexts (like $ in programming) might be incorrectly highlighted
  • Very uncommon currency formats or symbols might not be recognized
  • Currency values embedded in complex text without proper spacing might not be detected
  • The extension might not detect currencies in images or non-text elements
