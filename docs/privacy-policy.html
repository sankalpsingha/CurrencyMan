CurrencyMan Privacy Policy

Last Updated: March 30, 2025

CurrencyMan is designed with privacy in mind. This privacy policy explains what information our extension collects, how it's used, and how we protect your privacy.

Information Collection and Use

CurrencyMan collects minimal data:

  • User preferences: We store your preferred target currency locally on your device using Chrome's storage API. This information never leaves your browser.
  • Exchange rates: We cache currency exchange rates temporarily (24 hours) to minimize API calls. This data is stored locally on your device.
  • API usage: When you select text containing a currency value, we make an API call to fetch current exchange rates. No personal information is sent with these requests.

What We Don't Collect

We do not collect, store, or transmit:

  • Browsing history
  • Personal information
  • The content of websites you visit
  • Any user-identifiable information

The extension does not use cookies, analytics, or tracking technologies.

Third-Party Services

CurrencyMan uses the following third-party service:

  • Currency API: We use a free currency conversion API (https://cdn.jsdelivr.net/npm/@fawazahmed0/currency-api) to fetch exchange rates. When you select text containing a currency value, a request is made to this API. No personal information is included in these requests.

Data Security

All data is stored locally on your device. The extension uses HTTPS for all API requests to ensure secure data transmission.

Changes to This Privacy Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.
