Domain-Specific Settings

Currency symbols like "$" can represent different currencies depending on the website (USD, CAD, AUD, etc.). Domain-specific settings allow you to specify which currency should be used for particular websites.

1
Go to the Main Settings tab and scroll down to the "Domain-Specific Currency Settings" section
2
Add a domain mapping:
  • Enter a domain (e.g., "amazon.ca")
  • Select the currency to use for that domain (e.g., "CAD")
  • Click "Add Domain Mapping"
3
Use the "Use Current" button to automatically fill in the domain of the website you're currently on

Example: If you add a mapping for "amazon.ca" to use "CAD", then "$" symbols on amazon.ca will be interpreted as Canadian Dollars instead of US Dollars.
