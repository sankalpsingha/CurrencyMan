# CurrencyMan Icons

This directory should contain the following icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

To generate these icons:
1. Open the icon-generator.html file in your browser
2. Click the "Generate All Icons" button
3. Download each icon by clicking the "Download" links
4. Save the icons to this directory

The icons are required for the Chrome extension to display properly in the browser toolbar and extension management page.
